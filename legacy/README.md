# Legacy launchers

These batch files are kept for older dataset-building and synthetic-data workflows.

The preferred workflow for real scorecard development is now the `MCAV2_BatchProcessor` app because it supports:

- application data mapping
- duplicate AppID de-duplication
- all / paid / not-paid JSON uploads
- business credit report PDF matching and bureau signal extraction
- saved runs
- calibration exports

Use these legacy launchers only when you intentionally need the older fixed-folder workflow or synthetic test data.
